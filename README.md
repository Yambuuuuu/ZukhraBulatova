# ZB Project

Описание проекта будет добавлено позже.

## Структура проекта

```
├── assets/         # CSS, изображения и другие ресурсы
├── components/     # Vue компоненты
├── layouts/        # Шаблоны страниц
├── pages/          # Vue страницы (маршруты генерируются автоматически)
├── static/         # Статические файлы
└── nuxt.config.js  # Конфигурация Nuxt.js
```

## Установка

```bash
# Клонировать репозиторий
git clone <URL вашего репозитория>

# Перейти в директорию проекта
cd ZB

# Установить зависимости
npm install
```

## Использование

```bash
# Запустить сервер для разработки с горячей перезагрузкой
npm run dev

# Собрать проект и запустить сервер
npm run build
npm run start

# Генерировать статический проект
npm run generate
```
